# 🌿 AyurGenius – Ayurvedic RAG Chatbot
An intelligent **Retrieval-Augmented Generation (RAG)** chatbot that blends **ancient Ayurvedic knowledge with modern AI** to provide personalized health insights, herbal remedies, and lifestyle recommendations.

## 🚀 Overview
AyurGenius is designed to deliver **context-aware Ayurvedic guidance** using a structured medical dataset and NLP-powered retrieval techniques. It bridges traditional healthcare systems with cutting-edge AI to create a **smart, explainable, and holistic health assistant**.

## 🧠 Key Features

* 🔍 **RAG-Based Architecture**
  * Combines retrieval + generation for accurate responses
* 🌿 **Ayurvedic Intelligence**
  * Suggests diagonosis based on the duration, medical history, sleep patterns, etc.,. 
* 📊 **Rich Medical Dataset**
  * Covers diseases, symptoms, diagnosis, and treatments
* 🧘 **Lifestyle Recommendations**
  * Includes yoga, diet, and daily routine suggestions

## 📂 Dataset Description

The system is powered by a comprehensive dataset containing:

* Disease names (multi-language support)
* Symptoms & severity levels
* Diagnosis & medical tests
* Risk factors & environmental conditions
* Ayurvedic herbs & formulations
* Diet & lifestyle recommendations
* Yoga & therapy suggestions
* Medical interventions & prognosis

📌 Example insights include cough, diabetes, hypertension, migraine, and more with detailed Ayurvedic mappings 

## 🏗️ Project Structure
Ayurvedic_Rag_chatbot.ipynb  # Google Colab implementation

## ⚙️ How It Works

1. 📥 User inputs a health-related query in the code
2. 🔎 Retriever fetches relevant Ayurvedic knowledge
3. 🧠 Embeddings convert text into vector space
4. 🤖 LLM generates context-aware response
5. 🌿 Output includes:
   * Symptoms explanation
   * Herbal remedies
   * Diet & yoga suggestions
